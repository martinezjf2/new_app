Vacation Planner

Vacation Planner is a web application where users can input their vacations and save them so that way you vacation will not be forgotten. This is like a blog/reminder. You can change vacations, and you can check other users' vacation's as well to give you an idea where else you can discover.

Getting Started

Run Shotgun within the terminal, and it will direct you to the web application
